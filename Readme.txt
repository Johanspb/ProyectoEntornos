PROJECT TITLE: Greenfield
PURPOSE OF PROJECT:GreenField está diseñada para facilitar la gestión de huertos, proporcionando a los
usuarios una herramienta integral para planificar, monitorear y mantener sus cultivos de
manera eficiente. Los usuarios de GreenField se pueden incluir en: Personas que quieren
comenzar a cultivar en casa y necesitan orientación sobre cuándo y cómo sembrar,
aquellos que practican la agricultura en espacios reducidos, como balcones o patios,
profesores que desean enseñar sobre la biología de las plantas y la sostenibilidad
mediante huertos escolares, personas interesadas en cultivar sus propios alimentos.
GreenField no solo es una herramienta para la gestión de huertos, sino un recurso integral 
para empoderar a los usuarios a cultivar su propia comida y promover prácticas
sostenibles.

HOW TO START THIS PROJECT: 
para inciar este proyecto hay que tener instalada la base de datos  y colocar el puerto el nombre
de la base de datos el archivo keys.ts correspondientes a donde se instalo la base de datos,
intalar en la carpeta serve npm install y en la capeta client npm install
